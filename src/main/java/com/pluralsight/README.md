# Car Dealership App

## Overview
This Java console application allows users to view, search, add, and remove vehicles from a dealership inventory.

## Features
Create Vehicle class
Build SalesContract class
Save dealership data with FileManager
Create README.md
Prepare project for YearUp submission
## How to Run
1. Open the project in **IntelliJ IDEA**.
2. Make sure the file **`dealership.csv`** is located in the project folder.
3. Run the program by executing **`CarDealership.java`** (main method).

## Technologies Used
- **Java (OOP)**
- **IntelliJ IDEA**
- **CSV File Handling**
